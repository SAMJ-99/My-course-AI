## 📖 About

This repository contains four Artificial Intelligence assignments developed using Python.

The projects demonstrate fundamental concepts of Machine Learning, Deep Learning, Data Analysis, Data Visualization and Time Series Forecasting using real-world datasets.

---

# 📂 Assignments

## 📊 Assignment 1 — Data Analysis

Perform exploratory data analysis using Python.

### Topics

- Dataset exploration
- Data cleaning
- Statistical analysis
- Data visualization
- Correlation analysis

### Libraries

- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📈 Assignment 2 — Linear Regression

Build a Linear Regression model to predict numerical values.

### Topics

- Feature selection
- Train/Test Split
- Linear Regression
- Prediction
- Model Evaluation

### Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 🤖 Assignment 3 — LSTM Time Series Prediction

Develop an LSTM neural network capable of predicting future values from historical data.

### Topics

- Time Series Forecasting
- Data Scaling
- Sequence Creation
- LSTM Networks
- Model Training
- Prediction

### Technologies

- TensorFlow
- Keras
- MinMaxScaler

---

## 🚀 Assignment 4 — Rocket Lab Budget Analytics

Analyze Rocket Lab's historical budget and employee growth while forecasting future budget values.

### Features

- Budget Analysis
- Employee Analysis
- Correlation Matrix
- Budget per Employee
- Data Interpolation
- LSTM Prediction
